# Governance

## Project Leadership

William Murray has final authority on project direction, architecture decisions, and contribution acceptance for all SpeyTech open source projects.

## Decision Making

- **Technical decisions** are made based on alignment with project goals: determinism, correctness, and certification readiness.
- **Contributions** are reviewed for quality, test coverage, and adherence to project standards.
- **Disputes** are resolved by the project lead.
